# 好的短句
- **表示迫切做**

$$\begin{cases}
    \text{cry out for sth}\\
\end{cases}$$

- **站队**

<center>take sides in</center>

- **做。。。没意义**
  
  $$\begin{cases}
    \text{there is little/no point in doing sth}\\
\end{cases}$$

- **将要做/预计做某事**

$$\begin{cases}
    \text{due to do sth/due for sth}\\
    \text{be going to do sth}\\
\end{cases}$$
